# Raspberry_Pi_Server
	Raspberry_Pi_Server is simple server for raspberry-pi include user interface.
	
	This consists of Node.js, Express.js, jQuery and Bootstrap. 


## Installation & Run

	Clone this repository
	$ npm install
	$ sudo npm start
